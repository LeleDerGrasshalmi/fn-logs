# FN Logs

This project aims to upload and analyze certain parts about fortnite logs.

What we are interested in:

- API Errors
- MMS Errors
- Profile Versions

## Requirements

- node `18.19.1 >=`
- yarn `1.22.19 =>`
- AWS S3 / CF R2 Bucket to store logs

## Local Development

```bash
yarn dev

# or with new browser tab
yarn dev --open
```

## Building

```bash
yarn build
```
